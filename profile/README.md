# Generative Layers

[![Maven Central](https://img.shields.io/maven-central/v/com.generativelayers/generative-layers-core?color=blue&label=Maven%20Central)](https://central.sonatype.com/artifact/com.generativelayers/generative-layers-core)
[![License](https://img.shields.io/badge/License-Apache%202.0-green.svg)](https://www.apache.org/licenses/LICENSE-2.0)
[![Website](https://img.shields.io/badge/Website-generativelayers.com-blueviolet)](https://www.generativelayers.com)

A governance middleware framework for BDI agents that use Large Language Models. Generative Layers places a **safety boundary** between autonomous agent programs and external generative resources — enforcing policy, validation, auditability, and deliberative adoption.

## Quick Install

```xml
<dependency>
    <groupId>com.generativelayers</groupId>
    <artifactId>generative-layers-core</artifactId>
    <version>0.1.0</version>
</dependency>
```
