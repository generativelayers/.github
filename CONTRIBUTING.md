# Contributing

Thank you for your interest in Generative Layers.

## Scope

Contributions should support the project goal: governed generative resource layers for BDI agent systems, especially ASTRA, Jason, JaCaMo, and CArtAgO.

## Contribution types

- Framework code
- ASTRA examples
- Jason examples
- Documentation improvements
- Test cases
- Research notes linked to implementation

## Pull requests

Before opening a pull request, make sure that:

- The change has a clear purpose.
- Existing functionality is not broken.
- Tests or runnable examples are included where relevant.
- Documentation is updated when behaviour changes.

## Style

Keep the separation clear between symbolic BDI reasoning, external generative resources, governance checks, and execution-time admissibility.
